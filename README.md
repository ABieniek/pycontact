# pyContact
Tools for contact analysis of biomolecules

## Software requirements
* latest version of [VMD](http://www.ks.uiuc.edu/Development/Download/download.cgi?PackageName=VMD)
* Python 3.5
    * Modules:
        * PyQt5
        * SciPy (numpy and matplotlib)
